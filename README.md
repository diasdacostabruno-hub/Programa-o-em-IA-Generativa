# 🚀 Curso de Programação em IA Generativa

Bem-vindo ao repositório oficial do curso **Programação em IA Generativa**.

Este repositório foi criado para organizar todo o conteúdo do curso, incluindo aulas, materiais complementares, exercícios, projetos práticos e recursos utilizados ao longo da formação.

---

# 📚 Objetivo do Curso

O curso tem como objetivo ensinar os fundamentos e aplicações práticas da Inteligência Artificial Generativa, abordando desde conceitos básicos até a criação de aplicações modernas utilizando modelos de IA.

Ao final do curso, você será capaz de:

* Entender os fundamentos de IA e Machine Learning
* Trabalhar com Redes Neurais
* Utilizar técnicas de Processamento de Linguagem Natural
* Desenvolver aplicações com IA Generativa
* Criar modelos personalizados
* Integrar APIs de IA em projetos reais

# 🔧 Instalação

## 1. Clone o repositório

```bash
git clone https://github.com/seu-usuario/curso-ia-generativa.git
```

## 2. Entre na pasta do projeto

```bash
cd curso-ia-generativa
```

## 3. Crie um ambiente virtual

### Linux/macOS

```bash
python -m venv venv
source venv/bin/activate
```

### Windows

```bash
python -m venv venv
venv\Scripts\activate
```

---
